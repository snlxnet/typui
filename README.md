Status: in development, not really ready yet

## Usage
1. clone the dist directory
2. `node server.js`
3. copy the `Caddyfile` to wherever you need it and adjust the root path
4. `caddy run`

## Building
```shell
npm install
npm run build
```

## Features
- number inputs
- text inputs
- dynamic page size
- multi-page
- checkboxes
- focus tracking
- dyn guess on data type

## Missing Features

- proper font support
- swap button
- renaming the fields object
- hot reloading
- increment / decrement counters
- browser-only version
- radio buttons - can't come up with a DX I like
