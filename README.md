Status: Getting rid of the weird syntax and system variables

## Usage
1. clone the dist directory
2. `node server.js`
3. copy the `Caddyfile` to wherever you need it and adjust the root path
4. `caddy run`

## Building
```shell
npm install
npm run build
```

## Features
- number inputs
- text inputs
- dynamic page size
- multi-page
- checkboxes
- focus tracking
- dyn guess on data type
